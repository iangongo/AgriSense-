## Installation

To run AgriSense locally, follow these steps:

1. **Clone the repository:**

   ```bash
   git clone <repository-url>
   cd agrisense
2. cd agrisense-backend
   npm install
3. cd ../agrisense-frontend
   npm install
4.cd ../agrisense-backend
  npm start
5.cd ../agrisense-frontend
  npm start
